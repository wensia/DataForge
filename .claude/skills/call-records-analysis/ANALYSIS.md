# 通话质量分析指南

## 分析维度概览

```
通话质量分析
├── A. 销售话术（40分）
│   ├── 开场白
│   ├── 产品介绍
│   ├── 异议处理
│   └── 促成技巧
├── B. 服务质量（35分）

│   ├── 问题解决
```

## A. 销售话术分析

### 1. 开场白分析

**分析要点**：
- 是否有问候语（您好、你好）
- 是否自我介绍（机构名称、个人身份）
- 是否说明来意（看到咨询、给孩子了解课程）
- 是否确认对方身份（家长、是否本人）

**正向关键词**：
```
您好、你好、家长、瑞满分、瑞玛森、行未来、
我是、老师、课程顾问、了解、咨询、看到您
```

**负向表现**：
- 直接推销不问候
- 不说明身份和来意
- 开场过于冗长（超过30秒还没进入正题）

**示例评价**：
- 优秀："您好家长，我是瑞满分的课程顾问小李，看到您在我们这边咨询了关于孩子数学辅导的问题，请问是您本人吗？"
- 不合格："喂，是XX家长吗？我给你介绍下我们的课程..."

### 2. 产品介绍分析

**分析要点**：
- 是否先了解客户需求（孩子年级、科目、当前成绩、学习问题）
- 是否针对性介绍（根据孩子情况推荐）
- 是否突出卖点（师资、效果、特色）
- 是否有案例支撑（其他学员提升案例）

**正向关键词**：
```
您孩子、几年级、哪个科目、现在成绩、
针对、适合、我们的老师、师资、提升、效果、
一对一、小班、个性化、比如有个学员
```

**负向表现**：
- 不了解需求就推销
- 介绍过于笼统（"我们课程很好"）
- 夸大承诺效果

**需求挖掘问题示例**：
- "您孩子现在几年级了？"
- "主要是哪个科目需要提升？"
- "现在大概是什么成绩水平？"
- "平时学习中主要遇到什么问题？"

### 3. 异议处理分析

**分析要点**：
- 是否倾听客户异议（不打断）
- 是否表示理解（共情）
- 是否合理解释
- 是否有效转化

**正向关键词**：
```
理解、确实、您说的对、其实、这样、您看、
没关系、可以理解、是这样的
```

**负向关键词**：
```
不是的、你错了、必须、肯定不是这样、
你不懂、你听我说
```

**常见异议及应对**：
| 异议类型 | 示例 | 期望应对 |
|---------|------|---------|
| 价格异议 | "太贵了" | 理解 → 解释价值 → 对比效果 |
| 时间异议 | "没时间" | 理解 → 灵活安排 → 强调重要性 |
| 效果疑虑 | "没用吧" | 理解 → 案例说明 → 承诺保障 |
| 考虑考虑 | "再想想" | 理解 → 挖掘顾虑 → 邀约体验 |

### 4. 促成技巧分析

**分析要点**：
- 是否有邀约动作
- 是否确定下一步
- 是否留下联系方式

**正向关键词**：
```
试听、体验、来看看、约个时间、什么时候方便、
这周六、下周、可以先来、安排一下
```

**有效促成示例**：
- "您看这周六下午方便带孩子来试听一下吗？"
- "我帮您预约一节免费的体验课，您看哪天有空？"
- "要不我加一下您微信，把详细资料发给您看看？"

## B. 服务质量分析

### 1. 礼貌用语分析

**正向关键词**：
```
您、请、谢谢、麻烦您、好的、不客气、
感谢、辛苦、抱歉、不好意思
```

**统计指标**：
- 敬语使用频率："您" vs "你" 的使用比例
- 礼貌词频次：请、谢谢等出现次数

**负向表现**：
- 全程使用"你"而非"您"
- 无任何礼貌用语
- 命令语气（"你必须..."、"你要..."）

### 2. 响应质量分析

**分析要点**：
- 是否准确理解问题
- 回答是否专业完整
- 是否有效沟通

**正向表现**：
- 复述确认客户问题
- 分点清晰回答
- 主动补充相关信息

**负向表现**：
- 答非所问
- 回答模糊不清
- 遗漏关键信息

### 3. 问题解决分析

**分析要点**：
- 是否主动挖掘需求
- 是否提供解决方案
- 客户问题是否得到回应

**正向表现**：
- 主动询问深入了解
- 提供针对性建议
- 确认是否解决疑问

**负向表现**：
- 被动等待客户提问
- 只说不解决
- 推诿问题

### 4. 情绪管理分析

**分析要点**：
- 语气是否积极
- 面对困难客户是否保持平和
- 是否能安抚客户情绪

**正向表现**：
- 语气热情友好
- 耐心解答重复问题
- 能化解客户不满

**负向表现**：
- 语气生硬
- 表现不耐烦
- 与客户起冲突

## C. 合规检查

### 1. 话术规范检查

**检查项**：
- 是否按标准流程沟通（开场 → 需求挖掘 → 介绍 → 异议处理 → 促成）
- 信息是否准确（价格、课程、师资）
- 表达是否专业

### 2. 禁止用语检测

**禁止用语清单**：

| 类型 | 禁止用语示例 |
|-----|-------------|
| **绝对承诺** | 保证提分、一定能、肯定可以、百分之百、绝对 |
| **贬低竞品** | XX机构不行、其他都是骗人的、只有我们最好 |
| **泄露隐私** | 提及其他客户姓名、成绩、联系方式 |
| **不当表达** | 粗话、口头禅过多（嗯嗯嗯、那个那个） |
| **压迫销售** | 今天不报就没了、必须现在决定 |

## 转写文本格式化

### 原始格式（JSON）
```json
[
  {"start_time": 0.0, "end_time": 2.5, "speaker": "staff", "text": "您好..."},
  {"start_time": 2.8, "end_time": 5.2, "speaker": "customer", "text": "你好..."}
]
```

### 分析格式
```
【员工】(0.0-2.5s) 您好，我是瑞满分的课程顾问...
【客户】(2.8-5.2s) 你好，我想了解一下...
【员工】(5.5-10.2s) 好的，请问您孩子现在几年级...
```

### 简洁格式
```
员工：您好，我是瑞满分的课程顾问...
客户：你好，我想了解一下...
员工：好的，请问您孩子现在几年级...
```
