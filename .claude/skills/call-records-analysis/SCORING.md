## A. 销售话术（40分）

### 1. 开场白（10分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 9-10 | 问候语 + 自我介绍 + 说明来意 + 确认对方身份 |
| 良好 | 7-8 | 问候语 + 自我介绍 + 说明来意 |
| 合格 | 5-6 | 有基本问候和自我介绍 |
| 不合格 | 0-4 | 缺少基本问候或自我介绍 |

**评分要点**：
- 问候语：您好、你好
- 自我介绍：机构名称（瑞满分/瑞玛森/行未来）、个人身份
- 说明来意：看到咨询、了解课程需求
- 确认身份：是否家长、是否本人

### 2. 产品介绍（15分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 13-15 | 先了解需求，针对性介绍，突出卖点，有案例 |
| 良好 | 10-12 | 能根据客户情况介绍产品特点 |
| 合格 | 7-9 | 有产品介绍但较生硬 |
| 不合格 | 0-6 | 无介绍或不了解需求就推销 |

**评分要点**：
- 需求挖掘：孩子年级、科目、当前成绩、学习问题
- 针对性介绍：根据情况推荐合适课程
- 卖点呈现：师资、效果、特色、教学方式
- 案例支撑：类似学员提升案例

### 3. 异议处理（10分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 9-10 | 倾听异议、表示理解、合理解释、有效转化 |
| 良好 | 7-8 | 能回应异议，有解释 |
| 合格 | 5-6 | 简单回应异议 |
| 不合格 | 0-4 | 忽视异议或与客户争辩 |

**评分要点**：
- 倾听：让客户说完，不打断
- 共情：表示理解（"我理解"、"确实"）
- 解释：针对疑虑给出合理说明
- 转化：将异议转为机会

### 4. 促成技巧（5分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 4-5 | 有明确邀约，确定下一步行动 |
| 良好 | 3 | 有邀约意向 |
| 合格 | 2 | 提及后续联系 |
| 不合格 | 0-1 | 无任何促成动作 |

**评分要点**：
- 邀约：试听、体验、来访
- 确定时间：什么时候方便
- 下一步：明确后续安排

## B. 服务质量（35分）

### 1. 礼貌用语（10分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 9-10 | 全程敬语、称谓得体、有结束语 |
| 良好 | 7-8 | 大部分使用敬语 |
| 合格 | 5-6 | 有基本礼貌用语 |
| 不合格 | 0-4 | 缺乏礼貌或有不当用语 |

**评分要点**：
- 敬语使用：您、请、谢谢、麻烦您、好的
- 称谓：家长、老师（对客户）
- 结束语：感谢来电、再见

### 2. 响应质量（10分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 9-10 | 准确理解问题，回答专业完整 |
| 良好 | 7-8 | 能理解并回答问题 |
| 合格 | 5-6 | 基本能回应问题 |
| 不合格 | 0-4 | 答非所问或忽视问题 |

**评分要点**：
- 理解准确：正确把握客户意图
- 回答专业：信息准确、表达清晰
- 回答完整：不遗漏关键信息

### 3. 问题解决（10分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 9-10 | 主动挖掘需求，提供解决方案 |
| 良好 | 7-8 | 能解决客户提出的问题 |
| 合格 | 5-6 | 部分解决问题 |
| 不合格 | 0-4 | 未能解决问题 |

**评分要点**：
- 主动性：主动询问、挖掘深层需求
- 方案提供：针对问题给出建议
- 问题闭环：确认问题是否解决

### 4. 情绪管理（5分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 4-5 | 始终积极，能安抚客户情绪 |
| 良好 | 3 | 态度平和 |
| 合格 | 2 | 偶有波动但可控 |
| 不合格 | 0-1 | 有明显负面情绪 |

**评分要点**：
- 积极态度：语气热情、有耐心
- 情绪稳定：面对困难客户保持平和
- 安抚能力：能缓解客户不满

## C. 合规检查（25分）

### 1. 话术规范（15分）

| 等级 | 分数 | 标准 |
|-----|------|-----|
| 优秀 | 13-15 | 完全符合话术流程，表达专业 |
| 良好 | 10-12 | 基本符合话术规范 |
| 合格 | 7-9 | 有轻微偏离 |
| 不合格 | 0-6 | 严重偏离话术规范 |

**评分要点**：
- 流程规范：按标准流程沟通
- 信息准确：产品、价格等信息正确
- 表达专业：用词恰当、逻辑清晰

### 2. 禁止用语（10分）

| 情况 | 分数 |
|-----|------|
| 无禁止用语 | 10分 |
| 每出现一次 | -2分 |
| 最低 | 0分 |

**禁止用语清单**：

| 类型 | 示例 |
|-----|------|
| 绝对承诺 | `保证提分`、`一定能`、`肯定可以`、`百分之百` |
| 贬低竞品 | `XX机构不行`、`其他都是骗人的`、`只有我们` |
| 泄露隐私 | 提及其他客户姓名、成绩等信息 |
| 不当表达 | 粗话、口头禅过多（频繁"嗯"、"那个"） |

## 扣分规则汇总

| 扣分项 | 扣分 | 说明 |
|-------|------|-----|
| 未自我介绍 | -3 | 开场白缺失自我介绍 |
| 未确认客户身份 | -2 | 未询问是否本人/家长 |
| 打断客户 | -2/次 | 频繁打断客户说话 |
| 使用禁止用语 | -2/次 | 见禁止用语清单 |
| 答非所问 | -3 | 未回应客户问题 |
| 无促成动作 | -3 | 通话结束无下一步安排 |
| 态度问题 | -5 | 表现出不耐烦或消极态度 |
| 信息错误 | -3 | 提供错误的产品/价格信息 |
| 语速过快 | -2 | 说话太快客户难以理解 |
| 沉默过长 | -2 | 长时间无回应（超过5秒） |

## 评分等级

| 等级 | 分数范围 | 说明 |
|-----|---------|-----|
| **A 优秀** | 90-100 | 可作为标杆案例，值得学习 |
| **B 良好** | 80-89 | 表现良好，可持续优化 |
| **C 合格** | 70-79 | 基本达标，需要改进 |
| **D 待改进** | 60-69 | 需要培训和指导 |
| **E 不合格** | <60 | 需要重点关注和辅导 |
